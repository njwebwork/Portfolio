Portfolio Project Overview

This is my personal portfolio website, built to showcase my web development work, skills, and experience. It is fully responsive, built with Bootstrap, and features smooth animations powered by AOS. It includes an interactive contact form, project gallery, and dynamic navigation linking to project detail pages.

Key Features:
  • Mobile-first and responsive design
  • Projects section with detail pages
  • Smooth animations using AOS
  • Clean and modern layout with Font Awesome icons
  • Contact form and social media integration
Tech Stack:
HTML CSS JavaScript Bootstrap AOS
